# ysws-sim
A python-based tui game about balancing school and tech hobbies. Create projects and complete them to earn points for ysws projects, with each project's points contributing to the end-of-game score! Ngl I think I had this idea because I watched too much Community. I used AI minimally, only to troubleshoot if I couldn't work it out, and I used VSC because you can't run things on vim.
